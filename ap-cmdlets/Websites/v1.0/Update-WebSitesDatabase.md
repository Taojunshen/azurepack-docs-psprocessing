# Update-WebSitesDatabase

## SYNOPSIS
Updates a website database.

## SYNTAX

```
Update-WebSitesDatabase [[-HostingConnectionString] <String>] [[-MeteringConnectionString] <String>] [-Force]
 [-PerformPostUpdate] [-Type] <DatabaseType> [-RemoteSettings <RemoteSettings>] [-SuppressRequestIdLine]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
The **Update-WebSitesDatabase** cmdlet updates a website database.
Updating a hosting or metering database is usually performed after the controller has been updated.
This helps to synchronize the database with the updated controller.

## EXAMPLES

### Example 1: Update a hosting database
```
PS C:\> Update-WebSitesDatabase -Type "Hosting"
```

This command updates the hosting database.
